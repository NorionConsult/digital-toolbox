A practical overview of digital tools

Explore and engage in Circular Economy practices

This digital toolbox provides a structured overview of digital tools that support the transition into circular economy practices for SMEs. The toolbox is organized into six modules, sector packages and a catalogue. Each module contains relevant introduction and structured content that presents selected tools that provide methods, guidance, and resources for a wider audience. Get started by exploring the contents below!

For a structured approach

The SME Journey

Each module can contain selected tools, descriptions, target users, links, and practical guidance on how the tools can be applied.

Module 1

Awareness

Introduces key CE and RECP concepts, terms, and models to understand the benefits, risks, and potential of CE, even without prior knowledge.

View module
Module 2

Diagnose

Provides tools to assess CE adoption, identify impacts and gaps, perform hotspot analysis, and set a baseline.

View module
Module 3

Options

Offers tools to support innovation and design processes, review practices, and identify relevant strategies to address diagnosed hotspots.

View module
Module 4

Business Case

Brings tools which support the transformation from linear to circular business models and the identification of risks, relevant barriers, and potential gains.

View module
Module 5

Implement

Tools that supports the identification of key value chain partnerships, circular solutions or practices to test, and help move ideas into the testing phase.

View module
Module 6

Monitor

Selected tools for ongoing monitoring of solutions, including data gathering, selection of relevant indicators, and re-diagnosis after implementation.

View module

About the catalogue

Designed for easy navigation

The catalogue is structured to help SMEs quickly identify relevant digital tools. The modular format makes it possible to expand the site over time by adding new tools, categories, links, and case examples. This digital toolbox was developed as part of the EU4Green Recovery East Program, which seeks to facilitate the adoption of Circular Economy among SMEs in the countries of the Eastern Partnership (EaP).
